[gold][Command Docs][end]
[underline]/addAccount[end] ([underline]/add[end])
    -> Adds a new account to config.json and restarts RoSniper

[underline]/alias[end]
    -> Adds the ZSH alias <[underline]snipe[end]> to access RoSniper through the terminal
    -> Possible arguments for <[underline]snipe[end]> [bold](or RoSniper in general, this alias is NOT needed!)[end]:
        - No arguments to open the RoSniper menu
        - One user to snipe a singular user
        - Multiple users, seperated by commas (no spaces) to snipe multiple users
        - A / command to quickly run a command in RoSniper
            - Commands with arguments can also run through the alias!
        - [bold]-d[end] to enable Decline First Server
        - [bold]-m[end] to enable the monitoring-only mode
    -> If you change the location of your RoSniper executable, you will need to regenerate the alias.
    [bold]-> This command is only available on macOS systems. Command-line arguments are available on all supported operating systems.[end]

[underline]/changelog[end]
    -> Shows the RoSniper changelog

[underline]/cmds[end]
    -> Shows this window

[underline]/del[end] [RECENT_USER] | [RECENT_USER_INDEX] | *
    -> Deletes a specific user or all users from the Recent Users list

[underline]/delay[end] [SECONDS]
    -> Sets the delay between requests
        - Default: 0.01s
        - Current: [cur_delay]s

[underline]/donate[end]
    -> Opens a Roblox gamepass in your browser
    -> Possible arguments: 10, 50, 100, 200, 500, 1000, 2000, 5000, 10000
    -> [bold]Donating is not necessary to use RoSniper or any of its functionality, although I sincerely appreciate donations.[end]

[underline]/logout[end] (*)
    -> Removes current account or all accounts from config.json

[underline]/setRecents[end] [MAX_LENGTH] ([underline]/set[end])
    -> Sets the max length of Recent Users
        - Default: 5 users
        - Current: [cur_recent_users] users
        - Maximum: 99 users

[underline]/switch[end] [MAX_LENGTH] ([underline]/s[end])
    -> Switches accounts without re-authenticating with your cookie
    -> Possible arguments:
        - Nothing (type: None)
        - A valid cookie ID (type: int)
        - A valid username (type: str)

[underline]/toggleTips[end]
    -> Toggles if the tips on the Main Menu are shown
        - Default: True
        - Tips Shown: [cur_tips]

[gold][Temporary Commands][end]
The effects of these commands only last for one sniping session.

[underline]/declineFirst[end] ([underline]/df[end])
    -> Declines the first server of the [bold][Priority][end] user
        - Default: False
        - Current: [cur_df]

[underline]/monitoring[end] ([underline]/m[end])
    -> Monitors the status of a set of users without sniping them
        - Default: False
        - Current: [cur_m]

Commands in parentheses represent aliases.
Words in brackets represent values (ex: [bold][SECONDS][end] means [bold]the # of seconds[end])
